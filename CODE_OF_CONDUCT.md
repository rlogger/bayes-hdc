# Code of Conduct

This project follows the **Contributor Covenant, version 2.1**.

The full text is published at
[contributor-covenant.org/version/2/1/code_of_conduct](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
and is incorporated here by reference.

## Reporting

If you experience or witness behaviour that violates the Covenant, please
report it privately to the project maintainer:

- **rajdeeps@usc.edu**

Reports are confidential. The maintainer will acknowledge within 72 hours
and respond with next steps.

## Scope

The Covenant applies to all project spaces — the GitHub repository, issue
tracker, pull requests, discussions, and any official communication channel
— and to public spaces when an individual is representing the project.

## Enforcement

Maintainers may remove, edit, or reject contributions that violate the
Covenant, and may temporarily or permanently ban contributors whose
behaviour they deem inappropriate, threatening, or harmful. Enforcement
follows the Community Impact Guidelines in the Covenant.
